Simple Grade Calculator

Calculate total marks, average marks, and final grade for 5 subjects.

How it works

When this page loads, you will be asked to enter marks for 5 subjects using prompt boxes.

After entering all marks, the total, average, and grade will be displayed in an alert box.

Grade Scale

  • A+ : 90 and above
  • A : 80 to 89
  • B : 70 to 79
  • C : 60 to 69
  • D : 50 to 59
  • F : Below 50

Project Features

  • Marks are entered using prompt boxes
  • Total marks calculation
  • Average marks calculation
  • Grade calculation using if-else
  • Validation using while loop
  • Output using alert()

Built with HTML, CSS, Bootstrap and JavaScript
